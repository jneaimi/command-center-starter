# Weekly review

When invoked:

1. Read the recent notes in `knowledge/` and `inbox/`.
2. List any **orphans** — notes with no `[[wikilinks]]` in or out.
3. Propose **2–3 specific links** that would connect related notes, plus a short summary of what is in flight.
4. Present them for the user to approve. **Do not write or modify any file.**

This skill only reads and suggests. The user decides what to act on.
